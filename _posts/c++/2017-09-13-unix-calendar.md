# ps 명령어의 옵션

옵션 | 설명
---|---
u | 프로세스 사용자 이름과 시작 시간을 출력
f | 프로세스 정보를 한 줄로 자세히 출력
l | 프로세스 정보를 한 줄로 좀 더 길고 자세하게 출력
e | 환경에 대한 정보도 출력
a | 다른 사용자들의 프로세스들도 모두 표시
x | 로그인 상태에서 아직 완료되지 않은 프로세스들을 표시
r | 현재 실행 중인 프로세스들을 표시
j | 작업 중심 형태로 출력
