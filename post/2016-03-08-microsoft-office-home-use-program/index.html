Microsoft Home Use Program

Microsoft Home Use Program 을 아시나요?

보통 컴퓨터에서 가장 많이 사용하는 프로그램 중 하나가 바로 오피스 프로그램인데요, 10,800 원에 구매해서 사용할 수 있는 방법이 있어서 포스팅을 해봅니다.

바로 Microsoft Home Use Program 인데요, 줄여서 HUP라고 많이 부릅니다.

간단히 직장 이메일 인증만 되면 해당 이메일로 구매할 수 있는 링크를 보내줍니다. (물론 해당 직장이 MS와 라이센스 계약이 맺어져 있어야 하겠지만…)

Office 365처럼 매달 일정 금액씩 내는게 아니라, 한 번 구매하면 평생 쓸 수 있는 패키지 구매입니다. 저도 Office 2013 때 구매해서 잘 쓰다가 이번에 2016으로 다시 구매했습니다.

물론, 학생이라거나 선생님, 교직원 등이라면 Office 365도 무료로 쓸 수 있으니 그냥 Office 365를 사용하는 것도 방법입니다.
